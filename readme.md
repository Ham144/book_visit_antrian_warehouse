# RBAC

## Inisialisasi role saat user login dengan Active directory

`USER_ORGANIZATION`: jika user punya office _physicalDeliveryOfficeName_

`ADMIN_VENDOR` : jika user tidak punya office _physicalDeliveryOfficeName_

`DRIVER_VENDOR` : dibuat oleh `ADMIN_VENDOR`

`ADMIN_ORGANIZATION`: di set oleh `ADMIN_ORGANIZATION` yang sudah ada
